The 2026 Verizon Data Breach Investigations Report highlights AI bot traffic expanding at a 21% monthly rate, but Cequence's CQ Prime threat research team — contributors to four consecutive DBIR cycles — draws a critical distinction between CDN-layer observations and actual threats reaching backend APIs. CDN-level data captures surface-level internet activity similar to traditional web crawling, while the API layer, where dynamic transactions and authenticated interactions occur, reveals a substantially different threat landscape. Understanding this gap is essential for security teams allocating resources against bot-driven API attacks.
